The program uses a dictionary to store movie data and the `json` module to save and load information from a file.

## Features
- Add a new movie
- Edit movie information
- Delete a movie
- View all movies
- Search for a movie
- Save movie data to a JSON file
- Load movie data from a JSON file

## Technologies Used
- Python
- JSON

## How to Run the Program
1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the project folder in VS Code or another Python editor.
4. Run the Python file:
   vislas_PA.py
